# Proyecto VaR — Complementaria 2

Sistema computacional para calcular el **Value at Risk (VaR)** de un portafolio de activos financieros usando simulación histórica.

## Estructura del proyecto

```
proyecto_var/
├── data/               # Datos del portafolio y base de datos
│   ├── PORTAFOLIO1.xlsx
│   └── proyecto.duckdb
├── notebooks/          # Análisis y desarrollo en Jupyter
│   └── proyecto.ipynb
├── src/                # Código fuente principal
│   └── main.py
├── requirements.txt    # Dependencias del proyecto
├── pyproject.toml      # Configuración del proyecto
└── README.md
```

## Instalación

```bash
pip install -r requirements.txt
```

## Uso

Abre el notebook principal:

```bash
jupyter notebook notebooks/proyecto.ipynb
```
